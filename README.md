## ✨ Особенности

### 🔍 **Умный поиск**
Находите нужные материалы за секунды. Поиск работает по заголовкам, описаниям, тегам и авторам.

### 📊 **Удобная навигация**
- Фильтрация по типам контента (документация, блог, новости)
- Автоматически генерируемое оглавление в каждой статье
- Категории и теги для быстрого поиска смежных тем

### 🎨 **Современный интерфейс**
- Адаптивный дизайн для desktop и mobile
- Темная и светлая темы с автоматическим переключением
- Красивые баннеры для каждой статьи
- Полноэкранный режим для таблиц

### 📱 **Mobile First**
Полностью оптимизированная мобильная навигация с удобным нижним меню.

### ⚡ **Высокая производительность**
- Статическая генерация страниц (SSG) на Astro
- Оптимизированные изображения и код-сплиттинг
- SEO-оптимизация из коробки

## 🚀 Быстрый старт

### Для читателей

Просто откройте [hub.opensophy.com](https://hub.opensophy.com) и начните изучать материалы!

### Для разработчиков

Если хотите запустить проект локально или использовать код для своего хаба:
```bash
# Клонирование репозитория
git clone https://github.com/opensophy-projects/hub.git
cd hub

# Установка зависимостей
npm install

# Генерация документации из Markdown
npm run generate:docs

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```

## 📚 Как добавить свою статью

1. Создайте файл в папке `Docs/` с расширением `.md`
2. Добавьте frontmatter с метаданными:
```markdown
---
title: "Название статьи"
description: Краткое описание статьи для SEO и превью
type: docs # или blog, news
category: Категория
author: Ваше имя
date: 2025-01-17
tags: тег1, тег2, тег3
keywords: ключевые, слова, для, SEO
bannercolor: #3b82f6
bannertext: Текст на баннере
robots: index, follow
lang: ru
---

# Заголовок статьи

Содержимое...
```

3. Запустите `npm run generate:docs` для обновления индекса
4. Статья автоматически появится на главной странице!
