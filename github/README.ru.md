# Opensophy Hub (Русский)

## Слоган

**Реализовать Hub — значит превратить статическую документацию в живой продукт.**

[🇷🇺 Русский](./README.ru.md) | [🇬🇧 English](./README.en.md)

## В чем уникальность

У большинства конкурентов обычно компромисс:
- либо просто статичный docs-движок без "живых" блоков,
- либо SaaS-платформа с ограничениями по кастомизации и контролю.

**Уникальность Opensophy Hub:**
- static-first архитектура (контент в репозитории, деплой как статика),
- продвинутые интерактивные markdown-блоки,
- UX как у SPA, но без обязательного backend,
- live playground UI-компонентов прямо в документации.

То есть Hub позволяет делать **документацию, которая ведет себя как приложение**, но остается статическим проектом.

## Для чего проект

Opensophy Hub нужен для создания:
- документации продукта,
- технических гайдов,
- базы знаний,
- статей и учебных материалов.
