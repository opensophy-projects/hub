# Opensophy Hub (Русский)

> Современная платформа для публикации технических знаний в формате **статического сайта**.

[🇷🇺 Русский](./README.ru.md) | [🇬🇧 English](./README.en.md)

## Для чего проект

Opensophy Hub нужен для создания:
- документации продукта,
- технических гайдов,
- базы знаний,
- статей и учебных материалов.

Проект ориентирован на **static-first подход**: контент хранится в репозитории и публикуется как статический сайт.

## Демонстрация

- Основной сайт: https://opensophy.com
- Гайд по Markdown: https://opensophy.com/docs/markdown-guide/
- Гайд по UI-компонентам: https://opensophy.com/docs/component-guide/

## Основные фичи

- Расширенные таблицы из Markdown (фильтры, поиск, fullscreen)
- Карточки, stepper, alerts, графики через markdown-блоки
- UI playground для интерактивных компонентов
- Автоматическая навигация по файловой структуре `Docs/`
- Светлая/тёмная тема

## Запуск

```bash
npm install
npm run dev
```

Продакшен-сборка:

```bash
npm run build
```
