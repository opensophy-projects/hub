| `colors` | `[string,string]` | `['#A855F7','#06B6D4']` | Два цвета для двух потоков |

| `bend1` | `number` | `1` | Амплитуда изгиба первого потока |
| `bend2` | `number` | `0.5` | Амплитуда изгиба второго потока |
