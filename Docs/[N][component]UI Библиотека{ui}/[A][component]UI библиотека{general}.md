# UI Библиотека

**UI Библиотека** — каталог интерактивных React-компонентов для Opensophy Hub. Анимированные тексты, декоративные фоны, визуальные эффекты — каждый компонент представлен с живым превью и интерактивным настройщиком прямо в документации.

## Community компоненты

Это основа текущей библиотеки. Компоненты вдохновлены открытыми источниками и адаптированы под Hub: переработаны под TypeScript, получили универсальные пропсы для управления масштабом, цветом, анимацией и другими параметрами через интерактивный viewer.

Сейчас доступны две категории:

### Типографика

Компоненты для выразительной работы с текстом.

- **Блюрный текст** — появление по словам или буквам с эффектом размытия
- **Размытый текст** — случайный фаззинг строк через Canvas, глитч-режим
- **Морфинг текста** — плавный переход между словами через SVG blur-фильтр
- **Дешифрованный текст** — символы раскрываются из шума, эффект дешифровки
- **Градиентный текст** — анимированный градиент с yoyo, горизонтальным, вертикальным и диагональным движением
- **Сияющий текст** — световой блик скользит по тексту
- **Линейная тень текста** — анимированная диагональная штриховка в качестве тени
- **Волновой текст** — буквы покачиваются волной в бесконечном цикле
- **Перекатывающийся текст** — буквы перекатываются через 3D-вращение по оси X
- **Чередующийся текст** — автоматическая смена строк с stagger и spring-переходами
- **Круговой текст** — текст по кругу с реакцией на ховер
- **Зацикленный изогнутый текст** — бесконечный marquee по SVG-кривой, управляется перетаскиванием
- **Разбиение текста** — GSAP-анимация по символам, словам или строкам при скролле
- **Раскрытие при скролле** — слова появляются с opacity и blur при прокрутке
- **Всплывающий текст** — буквы всплывают снизу при скролле через GSAP ScrollTrigger
- **Скорость скролла** — бегущая строка, скорость которой зависит от прокрутки страницы
- **Вариативная близость** — буквы меняют параметры вариативного шрифта в зависимости от близости курсора
- **Скрытый текст** — текст скрыт облаком частиц, проявляется при наведении
- **Счётчик** — анимированный числовой счётчик с пружинной анимацией
- **Печатание текста** — эффект печатающей машинки с удалением и настраиваемым курсором

### Фон

Декоративные и интерактивные фоны на WebGL, Canvas и CSS.

- **Аурора** — шумовой шейдер с настраиваемыми цветовыми стопами
- **Мягкая аурора** — слоистая аурора на шуме Перлина, реагирует на мышь
- **Шелк** — волновой паттерн на Three.js с вращением и зерном
- **Иридесценция** — переливающиеся цвета, реакция на движение мыши
- **Плазма** — объёмная анимация, режимы forward, reverse и pingpong
- **Плазменная волна** — ray marching в 3D с двухцветным градиентом
- **Грейниент** — анимированный градиент с зерном, деформацией и ротацией
- **Цветовые изгибы** — анимированные цветовые полосы с warp-деформацией и интерактивом
- **Тёмная вуаль** — генеративный CPPN-шейдер, сканлайны, деформация пространства
- **Столп света** — ray marching на Three.js, интерактивный, с настройкой качества
- **Световые лучи** — лучи из заданной точки, следуют за мышью
- **Лучи** — анимированные объёмные лучи на модифицированном MeshStandardMaterial
- **Молния** — турбулентные молнии с настройкой цвета через HSV
- **Плавающие линии** — волновые линии с реакцией на мышь и параллакс-эффектом
- **Поле точек** — интерактивная сетка точек с bulge-эффектом под курсором
- **Призма** — 3D-призма с вращением, свечением и шейдерным цветом
- **Пиксельный взрыв** — пикселизация с рябью по клику и liquid distortion
- **Призматический всплеск** — спектральные лучи из центра, несколько режимов анимации
- **Градиентные жалюзи** — полосатый градиент со spotlight и distort
- **Око Саурона** — анимация, следит за положением курсора

## Как устроены компоненты

Каждый компонент — отдельная папка в `src/features/ui-components/` с двумя файлами:

```text
src/features/ui-components/
└── my-component/
    ├── my-component.tsx   ← сам компонент
    └── config.json        ← имя, категория, props, controls
```

`config.json` описывает всё необходимое для viewer-а: название, описание, категорию, список пропсов с типами и диапазонами значений. После этого компонент автоматически появляется в навигации и в интерактивном просмотрщике.

Viewer поддерживает **универсальные параметры** — они работают для любого компонента без изменений в его коде:

:::cards[cols=3]

:::card
[title]Трансформации
[icon]move
Масштаб, смещение по X/Y, вращение по оси Z — всё без перекомпиляции.
:::

:::card
[title]Внешний вид
[icon]eye
Прозрачность, яркость, контраст, насыщенность, размытие — CSS-фильтры поверх любого компонента.
:::

:::card
[title]Цвет
[icon]palette
Оригинальный цвет компонента или переопределение: сплошной цвет с полноценным color picker.
:::

:::

## Как добавить компонент

:::steps
:::step[status=done] Создайте папку
Разместите компонент в `src/features/ui-components/my-component/`.
:::
:::step[status=done] Добавьте config.json
Опишите `id`, `name`, `category`, `props` и `specificProps`. Registry найдёт компонент автоматически.
:::
:::step[status=done] Запустите проект
`npm run dev` — компонент появится в UI библиотеке с живым превью и настройщиком.
:::
:::

Подробнее о добавлении компонентов — в [руководстве по компонентам](/ui/component-guide/).
