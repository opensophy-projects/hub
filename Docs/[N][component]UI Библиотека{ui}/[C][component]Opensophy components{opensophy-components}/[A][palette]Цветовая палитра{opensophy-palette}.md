[uic:opensophy-palette]

Палитра берётся из дизайн-токенов Hub: тёмная тема построена вокруг почти чёрного фона `#0a0a0a`, поверхностей `#111111/#1a1a1a`, белого акцента и полупрозрачных белых границ. Светлая тема использует тёплый серо-бежевый фон `#E8E7E3`, поверхности `#d8d7d3/#dddcd8`, чёрный акцент и полупрозрачные чёрные границы.
