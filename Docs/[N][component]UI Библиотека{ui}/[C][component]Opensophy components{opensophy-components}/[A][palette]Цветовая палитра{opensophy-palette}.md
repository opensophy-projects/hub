[uic:opensophy-palette]

Палитра не зависит от текущей темы интерфейса: текст и карточки внутри превью принудительно получают контрастные цвета. Тёмная тема: слой 1 `#0a0a0a`, слой 2 `#111111`, слой 3 `#1a1a1a`. Светлая тема: слой 1 `#E8E7E3`, слой 2 `#d8d7d3`, слой 3 `#dddcd8`.
