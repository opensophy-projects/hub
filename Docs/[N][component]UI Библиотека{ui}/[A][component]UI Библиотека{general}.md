# UI Библиотека

**UI Библиотека** — это коллекция интерактивных компонентов для Opensophy Hub: анимированные тексты, декоративные фоны, визуальные эффекты и другие элементы интерфейса, которые можно быстро просматривать, настраивать и использовать в проектах.

Библиотека создана как практичный каталог: каждый компонент представлен отдельной страницей с живым превью, описанием и возможностью увидеть поведение компонента прямо в документации.

---

## Что уже есть

Сейчас в библиотеке доступны **community-компоненты** — компоненты, основанные на открытых идеях, популярных визуальных паттернах и UI-решениях сообщества.

Они помогают быстро добавить на страницу:

- анимированную типографику;
- декоративные фоновые эффекты;
- интерактивные визуальные блоки;
- motion-эффекты для современных интерфейсов;
- готовые элементы для лендингов, документации и экспериментальных UI.

## Типы компонентов

Компоненты сгруппированы по категориям, чтобы их было проще искать и сравнивать.

### Типографика

Компоненты для работы с текстом: анимации появления, морфинг, сияние, скролл-эффекты, счётчики и другие способы сделать текст более выразительным.

### Фон

Декоративные и интерактивные фоны: лучи, ауроры, плазма, линии, градиенты и визуальные эффекты для создания атмосферы страницы.

---

## Community и Opensophy: чем отличаются

| Направление | Статус | Описание |
|---|---|---|
| **Community компоненты** | Доступны сейчас | Подборка компонентов и эффектов, вдохновлённых открытым UI-сообществом. |
| **Opensophy компоненты** | В разработке | Авторские компоненты Opensophy с собственным визуальным языком, UX-подходом и более глубокой интеграцией в Hub. |

Community-компоненты закрывают текущую потребность в готовых UI-эффектах. Opensophy-компоненты будут развиваться как отдельная линейка — более цельная, узнаваемая и ориентированная на долгосрочное использование внутри проектов Opensophy.

---

## Для чего использовать библиотеку

UI Библиотека пригодится, если нужно:

- быстро найти визуальный компонент для страницы;
- протестировать эффект перед внедрением;
- посмотреть, как компонент выглядит в контексте документации;
- собрать идеи для лендинга, статьи, промо-страницы или интерфейса;
- использовать Hub как живую витрину UI-решений.

---

## Планы развития

В ближайшем развитии библиотеки основной фокус будет на двух направлениях:

1. **Расширение community-раздела** — больше готовых компонентов, категорий и визуальных паттернов.
2. **Запуск Opensophy-компонентов** — постепенное добавление авторских компонентов, которые будут отражать стиль Opensophy и лучше интегрироваться с экосистемой проекта.

Пока Opensophy-компоненты находятся в разработке, текущая библиотека продолжит пополняться community-компонентами.
