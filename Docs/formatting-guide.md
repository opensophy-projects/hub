Это руководство демонстрирует все возможности форматирования текста, доступные в статьях Hub. Используйте его как справочник при написании собственных статей.

## Заголовки

Hub поддерживает шесть уровней заголовков:

# Заголовок первого уровня (H1)

## Заголовок второго уровня (H2)

### Заголовок третьего уровня (H3)

#### Заголовок четвертого уровня (H4)

##### Заголовок пятого уровня (H5)

###### Заголовок шестого уровня (H6)

## Параграфы и текстовое форматирование

Обычный параграф текста. Текст автоматически переносится на новую строку, если достигнут край контейнера. Между параграфами автоматически добавляется отступ.

Вы можете использовать **жирный текст** для выделения важной информации.

Также доступен *курсивный текст* для акцентирования.

Можно комбинировать: ***жирный курсив*** для особого выделения.

Для зачёркнутого текста используйте ~~такой синтаксис~~.

## Списки

### Маркированный список

- Первый элемент списка
- Второй элемент списка
- Третий элемент списка
  - Вложенный элемент 1
  - Вложенный элемент 2
    - Ещё более глубокая вложенность
- Четвёртый элемент списка

### Нумерованный список

1. Первый пункт
2. Второй пункт
3. Третий пункт
   1. Вложенный пункт 3.1
   2. Вложенный пункт 3.2
4. Четвёртый пункт

### Смешанный список

1. Первый основной пункт
   - Подпункт с маркером
   - Ещё один подпункт
2. Второй основной пункт
   - Маркированный элемент
   - Ещё элемент

## Код

### Встроенный код

Используйте `inline code` для выделения команд, переменных или коротких фрагментов кода прямо в тексте.

Примеры: `npm install`, `const variable = 'value'`, `127.0.0.1`

### Блоки кода

## Специальные возможности

### Оглавление

Hub автоматически генерирует оглавление на основе заголовков H2 и H3. Оно отображается в боковой панели на десктопе и доступно через кнопку в мобильной навигации.

### Баннеры статей

Каждая статья имеет настраиваемый баннер. Цвет и текст баннера задаются в frontmatter:

- `bannercolor`: HEX-цвет баннера
- `bannertext`: Текст, отображаемый на баннере

### Метаданные

Hub поддерживает полный набор SEO-метаданных:

- `title`: Заголовок статьи
- `description`: Описание для поисковых систем
- `keywords`: Ключевые слова
- `author`: Автор статьи
- `date`: Дата публикации
- `tags`: Теги для категоризации
- `robots`: Правила индексации

## Рекомендации по написанию статей

### Структура

1. Используйте только один заголовок H1 в начале (он берётся из frontmatter)
2. Организуйте контент с помощью H2 для основных разделов
3. Используйте H3-H4 для подразделов
4. Добавляйте горизонтальные линии между крупными секциями

### Читабельность

- Разбивайте длинные параграфы на короткие (3-5 предложений)
- Используйте списки для перечислений
- Выделяйте ключевые термины жирным шрифтом
- Добавляйте примеры кода там, где это уместно

### Изображения

- Размещайте изображения в папке `public/assets/`
- Используйте осмысленные имена файлов
- Всегда добавляйте alt-текст для доступности
- Используйте title для подписей к изображениям

### Код

- Всегда указывайте язык программирования для блоков кода
- Добавляйте комментарии в сложных примерах
- Используйте inline-код для команд и переменных в тексте
- Проверяйте работоспособность примеров кода

## Заключение

Теперь вы знаете все возможности форматирования в Hub. Используйте это руководство как шпаргалку при создании собственных статей.

Для создания новой статьи:

1. Создайте файл `.md` в папке `Docs/`
2. Добавьте frontmatter с метаданными
3. Напишите контент, используя Markdown
4. Запустите `npm run generate:docs` для индексации
5. Статья автоматически появится на главной странице

Удачи в написании статей!
