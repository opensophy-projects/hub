## Кто мы?

**Opensophy** — коллекция open-source решений для IT-специалистов. Проекты, инструменты, гайды и образовательные материалы о безопасности, инфраструктуре и современной разработке.

В **Hub**е собраны документация, статьи, гайды и новости по ключевым темам, которые написаны автором Opensophy — veilosophy и соавторами. Платформа построена на открытом стеке технологий и доступна как самостоятельное open-source решение для публикации контента.

[Демонстрация форматирования](https://hub.opensophy.com/docs/markdown-guide/) — [GitHub репозиторий](https://github.com/opensophy-projects/hub)

Используйте **Меню** чтобы менять тему сайта и просматривать статьи, или **Поиск** для быстрого поиска нужного материала.

## Наш подход

6 ключевых принципов, которыми мы руководствуемся в работе.

| Принцип | Описание |
|---|---|
| 🔓 **Открытость** | Фокус на open-source решениях и свободных инструментах |
| ⚡ **Практичность** | Гайды и ресурсы, которые можно применить сразу |
| 🛡️ **Безопасность** | Best practices и фокус на защиту данных |
| 🔄 **Актуальность** | Современные инструменты и свежая информация |
| 📦 **Готовые решения** | Подборки компонентов и шаблонов для быстрого старта |
| 📚 **Образовательный контент** | Статьи, гайды и практические примеры |

## Возможности платформы

:::cards[cols=3]

:::card
[title]Мощь Таблиц
[icon]table-2
Сортировка, мультиселект фильтры, поиск с подсветкой совпадений, скрытие колонок, fullscreen-режим, drag-to-scroll и HTML-контент в ячейках — всё из Markdown.
:::

:::card
[title]Богатые Markdown Блоки
[icon]layout-template
Карточки, колонки, степпер, диаграммы, алерты и многое другое в чистом Markdown — без MDX и плагинов. Работает из коробки.
:::

:::card
[title]Live UI Components
[icon]play
Интерактивный playground прямо в документации. Настраивайте scale, rotation, blur, gradient и скорость анимации для любого компонента в реальном времени.
:::

:::card
[title]Opensophy Style
[icon]palette
Лаконичный и современный дизайн с поддержкой тёмной и светлой темы, плавными переходами и mobile-first навигацией через bottom bar.
:::

:::card
[title]Структура = Папки
[icon]folder-tree
Папки в `Docs/` — это и есть навигация на сайте. Никакой настройки роутинга, сайдбар и breadcrumbs генерируются автоматически.
:::

:::card
[title]Ленивое заполнение
[icon]feather
Сложные Markdown-блоки пишутся в разы короче. Изображения, алерты и компоненты — без лишнего кода.
:::

:::
