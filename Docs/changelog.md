**Обновление Hub до V3.5**

Полный релизный обзор изменений относительно V3.4. В репозитории V3.4 сохранена в папке `v3.4`, а всё вне этой папки относится к новой версии.

### Главное
- Проект обновлён с версии `3.4.0` до `3.5`.
- Hub превратился из базовой документационной сборки в полноценный контент-хаб: добавлены разделы документации Hub, статьи opensophy, резюме и расширенная UI-библиотека.
- Добавлена система кастомных страниц, чтобы отдельные страницы могли рендериться как React-компоненты с собственными метаданными, а не только как Markdown-документы.
- UI-библиотека получила автоматическое обнаружение компонентов и исходников: компоненты больше не нужно вручную подключать в реестре, достаточно положить их в нужную папку.
- Обновлены дизайн-токены, тема, dev-панель, таблицы, навигация, поиск и генерация данных.

### Добавлено
- Новый раздел **«Документация Hub»**:
  - общее описание Hub;

### Изменено и улучшено
- Обновлена структура контента: вместо одного `welcome.md` теперь используется многоразделная документация с навигационными группами, категориями и статьями.
- Обновлён `welcome.md`: стартовая страница стала частью более крупной структуры Hub.
- Обновлены sitemap, manifest, robots и llm-файл под новую структуру страниц.
- Обновлён Layout и страницы Astro: добавлена поддержка новых маршрутов, кастомных страниц и нового отображения категорий/документов.
- Улучшена навигация и единая поисковая панель: они адаптированы под увеличившееся дерево документов и новые типы страниц.
- Улучшены компоненты отображения Markdown-контента: карточки, alert, code block, chart block, image card, overlay, stepper и not found приведены к новой теме и общему стилю.
- Улучшен парсер HTML/Markdown-контента и работа с таблицами.
- Улучшены table-компоненты: фильтры, control bar, modal и table view получили обновлённую тему и отдельные тесты для парсинга, фильтрации и копирования.
- UI viewer теперь умеет показывать не только компонент, но и его исходники из папки компонента.
- Компонентный registry стал устойчивее: выбирает `index`, preview-файл, основной файл и fallback-кандидаты в предсказуемом порядке.
- Dev bridge и генераторы документов/сайткарты обновлены под новую структуру контента и кастомные страницы.
- Тема вынесена из разрозненных значений в общую систему токенов, что упрощает поддержку dark/light режимов.
- Контакты и site-config обновлены под актуальные данные проекта.

### Совместимость и миграция
- V3.4 оставлена в папке `v3.4` как архив предыдущей стабильной версии.
- Новая версия находится в корне проекта.
- Для добавления UI-компонента теперь предпочтительно создавать папку компонента в нужной категории и документационный Markdown-файл, а не вручную расширять registry.
- Для уникальных страниц следует использовать механизм кастомных страниц с `Page.tsx` и `metadata.ts`.


### Полная инвентаризация V3.5 по коду
- **Routing и страницы:** динамический маршрут теперь собирает Markdown-документы, страницы категорий и custom pages; главная страница умеет переключаться между `welcome.md` и React-лендингом через `public/data/site-config.json`.
- **Custom pages:** добавлен слой `src/custom` с auto-discovery `Page.tsx`/`Page.jsx`, `metadata.ts`, slug из имени папки и поддержкой подключения через frontmatter `custom`.
- **Категории:** каждая `[C]` папка стала отдельной страницей категории с карточками документов, иконками, тегами, датами и parent-навигацией.
- **Markdown engine:** обновлена обработка карточек, колонок, steps, tabs, chart, math/KaTeX, alert-блоков и сокращённого синтаксиса изображений из `public/assets`.
- **SEO слой:** Layout теперь централизует canonical, robots, keywords, Open Graph, Twitter Cards, favicon из site-config, JSON-LD `WebSite` для главной и `Article` для статей.
- **Навигация:** sidebar/search адаптированы под nav popovers, категории, category pages, новые типы документов и большой manifest.
- **Таблицы:** таблицы получили отдельные утилиты парсинга, фильтрации и копирования, UI-тему, modal/fullscreen поведение, controls bar и тесты.
- **CodeBlock:** улучшены подсветка через highlight.js, поиск по коду, копирование, fullscreen/modal режим и вкладки кода.
- **Theme system:** добавлены theme tokens, CSS-переменные, overrides в localStorage, отдельные theme utils и hook `useTheme`.
- **Dev panel:** обновлены Docs/Assets/Site/Contacts панели, общий UI kit панели, toast bus, theme layer и dev bridge.
- **UI viewer:** добавлены универсальные настройки внешнего вида, lazy loading, source-code tab, preview приоритет и безопасный fallback при ошибках загрузки.
- **Публичные данные:** обновлены `manifest.json`, `sitemap.xml`, `llm.txt`, `robots.txt`, `site-config.json`, favicon и логотипы.
- **Качество:** добавлены unit-тесты для theme, storage, scroll utils, table parser/filter/copy/theme; добавлены custom test runner, eslint runner, LCOV writer и Sonar-конфигурация.
- **Документация V3.5:** обновлены Frontmatter guide и UI library guide, добавлены новые страницы Quickstart/Deploy и SEO/Indexing.

### Итог
V3.5 — крупный релиз, который расширяет Hub от статичного набора Markdown-страниц до полноценной платформы для документации, статей, кастомных React-страниц и визуальной UI-библиотеки с автоматическим обнаружением компонентов, исходниками, улучшенной темизацией и обновлённой инфраструктурой качества.
